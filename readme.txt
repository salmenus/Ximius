Ximius
======

An HTML5 drawing and animation framework, powerful and easy to use.

About
-----

Ximius is based on Html5's `canvas` capabilities and on OO javascript.
It provides an easy and powerful way to create animations for web sites 
and web-based mobile applications.

It's developed and maintained by Salmen Hichri 
`salmen (dot) hichri (at) gmail (dot) com`
